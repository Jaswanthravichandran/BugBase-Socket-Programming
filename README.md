## Installation

### Windows and Linux

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jaswanthravichandran/BugBase-Socket-Programming.git

2. **Navigate to the project directory:**
   ```bash
   cd BugBase-Socket-Programming

3. **Run the setup script:**
